# echo-production-example
Example App using golang, echo, sqlx, and so on.


# Requiment
+ mysql or mariadb

# Note
+ This application use go 1.11.
+ don't use glide or dep, use go modules.

set environment variables.
```
GO111MODULE=on
```

```
$ cd echo-production-example
$ go build
```


# Running
```
$ cd echo-production-example
$ go run main.go
```

# Running［fresh］
```
$ cd echo-production-example
$ fresh
```
